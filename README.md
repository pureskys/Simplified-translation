# 关于软件
这是一个翻译软件，中文名：极简翻译。曾上架于酷安市场；
软件采用Java安卓原生开发，体积小，性能高。
这个软件也是我第一次使用Android studio开发的第一个软件，软件的前身是我高中使用iapp开发的极简翻译，大学因为实在是无聊就使用Java全部重构了。
